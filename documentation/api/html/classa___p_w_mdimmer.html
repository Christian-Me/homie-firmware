Detailed Description

write a value to channel

PWM dimmer actuator plugin.

loop function of the PWM dimmer plugin

Parameters
datapointer to PropertyData
Returns
(boolean) true = achnkoleged new value

The PWM Plugin comes with it's own loop and timer for frequent updates during dimming events. If the timer is triggered it in- or decrements the _current value to reach the _target The limits are checked and the current value is fine tuned if the target is reached within the _increment range

Handles one PWM output including smooth fading with an given duration amd fade steps
