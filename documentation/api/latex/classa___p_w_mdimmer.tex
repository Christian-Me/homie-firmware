\doxysubsection{Detailed Description}
write a value to channel 

PWM dimmer actuator plugin.

loop function of the PWM dimmer plugin

The PWM \mbox{\hyperlink{class_plugin}{Plugin}} comes with it\textquotesingle{}s own loop and timer for frequent updates during dimming events. If the timer is triggered it in-\/ or decrements the {\ttfamily \+\_\+current} value to reach the {\ttfamily \+\_\+target} The limits are checked and the current value is fine tuned if the target is reached within the {\ttfamily \+\_\+increment} range

Handles one PWM output including smooth fading with an given duration amd fade steps \begin{DoxyRefDesc}{Todo}
\item[\mbox{\hyperlink{todo__todo000001}{Todo}}]logarithmic as LED for examples fades brightness on low duty cycles stringer than on higher values 

lower limits as motors for example don\textquotesingle{}t spin on low duty cycles (some motors can do slow rpm if you start them with a higher value and then step down) \end{DoxyRefDesc}
